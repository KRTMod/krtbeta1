# KRT 音频文件说明

本目录包含KRT轨道交通模组所需的音频文件。

## 音频文件格式
- 所有音频文件必须是OGG格式
- 推荐采样率：44.1kHz
- 推荐位深度：16位

## 音频文件目录结构
- `train/`: 列车相关音频
  - `run.ogg`: 列车走行音
  - `horn.ogg`: 列车鸣笛音
  - `brake.ogg`: 列车刹车音
- `door/`: 屏蔽门相关音频
  - `open.ogg`: 屏蔽门开门音
  - `close.ogg`: 屏蔽门关门音
- `announcement/`: 报站相关音频
  - `arrival.ogg`: 列车进站广播
  - `departure.ogg`: 列车报站广播
  - `station.ogg`: 列车到站广播

## 如何自定义音频
1. 准备符合格式要求的OGG音频文件
2. 将文件重命名为对应的名称
3. 替换本目录下的对应文件
4. 重新启动Minecraft使更改生效

## 注意事项
- 音频文件大小不宜过大，以免影响游戏性能
- 建议保持音频文件长度适中（5-15秒）
- 报站音频可以根据不同线路和车站定制化
